## About me

I am a Software Engineering student at FPT University Da Nang and a backend-focused developer with hands-on experience in building production-grade systems using **ASP.NET Core**, **Clean Architecture**, and cloud-native tools.

My main strength is designing backend systems that are not only functional, but also maintainable, testable, and ready for real-world usage. I care about clear architecture, reliable APIs, performance optimization, and smooth collaboration between backend, frontend, DevOps, and product logic.

I have worked across different parts of the software development lifecycle, including:

- Designing RESTful APIs and backend architecture
- Building authentication and authorization flows with JWT / OAuth 2.0
- Working with relational and NoSQL databases
- Optimizing API performance and database access
- Implementing real-time features with SignalR
- Handling asynchronous workloads with Redis, RabbitMQ, and background jobs
- Containerizing services with Docker and Docker Compose
- Setting up CI/CD workflows with GitHub Actions
- Integrating third-party services such as payment gateways, cloud storage, and AI pipelines

## My engineering mindset

I like to build software with three principles in mind:

### Clean structure first

I prefer systems with clear separation of concerns, where business logic, infrastructure, database access, and API layers are easy to understand and maintain.

### Performance matters

I enjoy finding bottlenecks in APIs, database queries, caching strategies, and deployment environments — then improving them with practical engineering decisions.

### Real-world delivery

I care about shipping software that works outside the local machine: tested, containerized, documented, and ready to be deployed.
